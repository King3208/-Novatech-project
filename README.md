# NovaTech Gadgets — Multi-Page Website

A modern, responsive multi-page website built with **HTML**, **CSS**, and **Vanilla JavaScript** — no frameworks, no libraries, no CDN dependencies.

Built by Cybersecurity and Datascience students (GROUP 4) at the **Federal University of Agriculture, Abeokuta (FUNAAB)** as a university group project, 2026.

---

## Project Structure

```
MiniProject/
├── index.html          — Home page
├── about.html          — About page
├── products.html       — Products catalogue with filter
├── faq.html            — FAQ accordion page
├── contact.html        — Contact form page
├── css/
│   └── style.css       — Single shared stylesheet (all styles)
├── js/
│   └── main.js         — Single shared JavaScript file (all behaviour)
└── images/
    ├── products/       — Product images (add your images here)
    └── team/           — Team member photos (add your images here)
```

---

## Features

- ✅ Sticky navbar with active page highlighting
- ✅ Hamburger menu for mobile (smooth slide-down transition)
- ✅ Scroll animations using IntersectionObserver (respects prefers-reduced-motion)
- ✅ Back-to-top button with fade-in/out
- ✅ Product catalogue with category filter (no page reload)
- ✅ Accordion FAQ (single-open invariant)
- ✅ Contact form with inline validation (no page reload)
- ✅ Fully responsive — works from 320px to 1920px
- ✅ Dark modern tech aesthetic (#1a1a2e background, #00d4ff accent)
- ✅ WCAG 2.1 AA contrast compliant body text

---

## How to Run

No build step required. Just open any HTML file directly in a browser:

